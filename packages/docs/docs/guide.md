# sexxion-x

一个 React Ui 组件库

技术栈：vite + react + typescript + dumi

#### 基本使用

##### 安装

安装组件库

```bash
$ npm i @sexxion-x/ui
```

#### 使用

```tsx
import React from 'react';
import { Button } from '@sexxion-x/ui';
import '@sexxion-x/ui/dist/style.css';

const App: React.FC = () => {
  return <Button>button</Button>;
};

export default App;
```
