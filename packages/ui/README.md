# @sexxion-x/ui

#### 文档地址

`https://codeusb.github.io/sexxion-design`

#### 安装依赖

```bash
$ npm i @sexxion-x/ui -S
```

#### 基本使用

```tsx
import React from 'react';
import { Button } from '@sexxion-x/ui';

const App: React.FC = () => {
  return <Button>button</Button>;
};

export default App;
```
